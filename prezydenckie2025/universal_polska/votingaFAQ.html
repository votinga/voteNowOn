IT not open source but for free. For free is use of smart contracts, you pay for ether which is using blockchain as your personal database but in public space.

Note SSN will be encrypted by keccack256 function or in plain if you put it that way on blockcahin.

There is many ways how you can aquaire ether or MATIC used by polygon chain – on which are initial smart contracts and start of this project.

Admins can transform state of smart contracts to change behaviour of those smart contracts to act differently to the data which you stored.

Before you click create profile or CreateUpgradableProfile

( the create profile use static smart contract -> unmutable , the upgrade able profile use beacon pointing to current implementation of profile set by administration of protocol mutable)

Step 1) Before Creating profile read this instruction below carefully and build your UniqueID:

Original profile should have unique ID

in our concept of initial anonimization of profiles fragile data.

Unique ID is encrypted with keccack256 and is fixed length hex number


How to build unique ID


Type in input field nex to encrypt button data as follow:

Type inside “USA” + State Code -see table below + your Social Security Number + YYYYMMDD - where YYYY is your year of birth, MM is your month of birth and DD is your day of birth. example USAILLNNNNNNN20001201 type similar thing just in,

NNN…NN length might vary might contain letters and remember that smal a or capital A letters give different encrypted hash code.

State codes to be used:

Illinois - ILL

New Jersey - NJ

New York - NY


If you lost keys and want to recreate profile after you have hash if it is duplicate rename last digits and letters with d1,d2,..d20 . This might be not broad available process in this first voting.

createUpgradeableProfile is mutable code form, where scripts of how profile can be used are updated automatically when protocol Votinga is updated.

Still with old static profile even after upgrade most functionalities will be kept, but to migrate to new protocol you will need to recreate and validate new profile.
